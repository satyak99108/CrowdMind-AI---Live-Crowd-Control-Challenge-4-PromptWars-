# CrowdMind-AI---Live-Crowd-Control-Challenge-4-PromptWars-

<h3>Step-by-Step Execution Guide</h3>
<h4>Prerequisites</h4>
The user must have Node.js (v18 or higher) and npm installed on their computer.

Step 1: Clone the Repository<br>
Clone the remote GitHub repository to their local computer:

<pre>git clone https://github.com/satyak99108/CrowdMind-AI---Live-Crowd-Control-Challenge-4-PromptWars-.git</pre>
<br>
Step 2: Navigate into the Project Folder
Move into the project directory:
<pre>cd CrowdMind-AI---Live-Crowd-Control-Challenge-4-PromptWars-</pre>
<br>
Step 3: Install Project Dependencies
Run npm install to download and install all required packages (react, react-dom, lucide-react, vite):
<pre>npm install</pre>
<br>
Step 4: Start the Development Server
Run the application in development mode:
<pre>npm run dev</pre>
<br>
Step 5: Open in Browser
Open any web browser (Chrome, Edge, Firefox, Safari) and navigate to:
<pre>http://localhost:3000</pre>
<br><br>
Optional: Production Build & Testing
If someone wants to generate an optimized production build:
<pre>npm run build</pre>
Preview Production Build:
<pre>npm run preview</pre>
